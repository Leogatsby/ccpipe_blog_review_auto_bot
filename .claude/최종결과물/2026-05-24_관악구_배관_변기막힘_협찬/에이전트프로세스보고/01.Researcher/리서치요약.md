## 키워드별 상세 분석

▶ 키워드: 봉천동 변기막힘
   검색량(월)  : PC 60 + 모바일 340 = 합계 400
   블로그 발행 : 총 180건 / 최근30일 4건
   트렌드      : 상승↑
   KGR 점수    : 0.01
   경쟁강도    : A (상위노출 유력)
   판단 근거   : 지역 특화 키워드, 공급 극히 적음
   콘텐츠 전략 :
     ✅ 제목에 '봉천동 변기막힘' 정확히 포함
     ✅ 긴급 상황 체험 후기 형식
     📈 장마철 접근으로 시즌성도 확보

▶ 키워드: 신림동 변기막힘 비용
   검색량(월)  : PC 80 + 모바일 420 = 합계 500
   블로그 발행 : 총 240건 / 최근30일 6건
   트렌드      : 유지→
   KGR 점수    : 0.012
   경쟁강도    : A (상위노출 유력)
   판단 근거   : 비용 키워드 + 지역 조합으로 전환율 극대화
   콘텐츠 전략 :
     ✅ 비용 구간표 필수 삽입
     ✅ 정찰제 강조로 신뢰 구축
     ✅ 비용 방어 패턴 적용

▶ 키워드: 서울대입구 변기 역류
   검색량(월)  : PC 40 + 모바일 180 = 합계 220
   블로그 발행 : 총 90건 / 최근30일 2건
   트렌드      : 유지→
   KGR 점수    : 0.009
   경쟁강도    : A (상위노출 유력)
   판단 근거   : 역류 키워드는 공포심 유발 → 교육형 글로 전문성 어필
   콘텐츠 전략 :
     ✅ 원인 설명 → 해결 과정 스토리텔링
     ✅ 장비(CCTV 카메라) 어필로 전문성 강조
