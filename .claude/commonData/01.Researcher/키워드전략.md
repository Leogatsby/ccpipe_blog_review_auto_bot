# 3. 키워드 전략 — 경쟁도별 공략

배관 업종의 특성상 문제 발생 시 즉시 검색하여 업체를 찾는 "결정 단계"의 독자가 많습니다.
단순 검색량만 높은 메인 키워드보다 실질적인 전환을 이끌어내는 롱테일 키워드에 집중합니다.

## 메인 키워드 (포화 — 10만+ 예상)
- 하수구막힘
- 배관뚫기
- 하수구뚫는법

→ **단독 공략 어려움. 롱테일로 우회.** 메인 키워드는 본문에 자연스럽게 섞어 쓰는 용도로만 활용합니다.

## 롱테일 키워드 (기회)
블로그의 핵심 타겟팅 키워드입니다. 독자 의도에 맞는 글 패턴을 적용하여 작성해야 합니다.

| 키워드 | 독자 의도 | 글 패턴 |
|---|---|---|
| 하수구막힘 비용 | 결정 (가격 비교 중) | 비용 방어 패턴 |
| 화장실 하수구 냄새 원인 | 관심 (아직 업체 안 찾음) | 원인-해결 교육 패턴 |
| 싱크대 막힘 자가수리 | 관심 (직접 해보려는 중) | 프로세스 공개 패턴 |
| 하수구 업체 선택 기준 | 비교 (업체 고르는 중) | Why 질문법 패턴 |
| 아파트 배관 막힘 | 결정 (지금 막혀서 급함) | 긴급 출동 신뢰 패턴 |
| 하수구 역류 원인 | 관심 (무서워서 검색) | 실패 사례 패턴 |
| 고압세척 배관청소 가격 | 결정 (견적 비교) | 비용 방어 패턴 |
| 오래된 아파트 배관 교체 | 비교 (큰 공사 고민 중) | 케이스 스터디 패턴 |
