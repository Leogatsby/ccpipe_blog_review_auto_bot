# 콘텐츠 캘린더 — 계절별 키워드

| 시기 | 키워드 | 이유 |
|---|---|---|
| 3~4월 | 봄맞이 배관 점검, 빙결 후 배관 파손 | 겨울 동파 후 수리 수요 |
| 6~8월 | 장마철 하수구 역류, 배수구 냄새 | 습기 + 집중호우 |
| 9~10월 | 추석 전 배관 점검 | 명절 전 집 정비 |
| 11~12월 | 동파 방지, 배관 보온 | 겨울 대비 |
| 연중 | 하수구막힘 비용, 업체 선택 기준 | 상시 수요 |
